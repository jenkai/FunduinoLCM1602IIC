FunduinoLCM1602IIC
==================

這是arduino透過LCM1602介面模組操作1602 LCD顯示器的範例

腳位定義
arduino   LCM1602
  +5V       VCC
  GND       GND
  A4        SDA
  A5        SCL
  
執行範例前請先安裝LiquidCrystal函式庫

將LiquidCrystal目錄整個放到

C:\Program Files (x86)\Arduino\libraries 之下

重新啟動arduino IDE後

再開啟LCM1602IIC 範列程式執行即可
